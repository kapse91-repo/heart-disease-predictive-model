# Heart Disease Predictive Modeling

This project is a supervised machine learning project developed for an internship task on predictive modeling. The goal is to predict the presence of heart disease using patient health data.

## Algorithms Used
- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

## Features
- Data preprocessing using pipelines
- Train-test split for model evaluation
- Classification reports
- Confusion matrices
- ROC curve and AUC comparison

## Dataset
The project uses a cleaned heart disease dataset stored as `cleanned.csv`.

## Results
The models were trained and tested on the dataset, and Random Forest achieved the best performance among the applied models.

## Files
- `Predictive_Modeling_Project.ipynb` – main notebook
- `cleanned.csv` – dataset
- `requirements.txt` – dependencies

## Tools & Libraries
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
